# 🕵️‍♂️ Adivinamon (Guess Who: Anime & Pokémon Edition)

A real-time multiplayer web game that brings the classic tabletop experience of "Guess Who?" (Adivina Quién) to the world of anime and Pokémon. Connect with a friend via a room code, hop on Discord, and deduce each other's secret character pick!

## ✨ Features

* **Real-Time Multiplayer:** Create or join a room using a unique 5-character code. State is synced instantly via Pusher.
* **Three Character Sources:**
  * **Templates:** Curated packs of anime characters from popular series. Mix and match multiple packs.
  * **Search Anime:** Search any anime by name via the Jikan API and play with its cast.
  * **Pokémon:** Classic Pokémon sprites from PokeAPI. Pick a generation (Gen 1–9) or play with all 1025.
* **Two Game Modes:**
  * **Classic Guess Who:** Each player secretly picks a character; ask yes/no questions to deduce the opponent's pick.
  * **Rule Master:** Each player writes a secret rule (e.g., "Characters with blue hair"). Take turns asking if characters match the opponent's rule, then guess the rule itself.
* **Smart Caching:** Powered by TanStack Query to minimize API calls and load data instantly.
* **Honor System UI:** Built to be played while chatting over Discord. You handle the talking, the app handles the board state.

## 🛠️ Tech Stack

* **Frontend:** [Next.js 16](https://nextjs.org/) (App Router) with React 19
* **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/)
* **Local State:** [Zustand](https://github.com/pmndrs/zustand)
* **Data Fetching:** [TanStack Query](https://tanstack.com/query/latest), [Jikan API](https://jikan.moe/) (anime), & [PokeAPI](https://pokeapi.co/) (Pokémon)
* **WebSockets / Multiplayer:** [Pusher](https://pusher.com/) (presence channels)

## 🎮 How to Play

1. **Host a Game:** Player 1 clicks "Create Room" and picks a character source — anime templates, anime search, or Pokémon.
2. **Join a Game:** Player 2 enters the 5-character Room Code generated by the Host.
3. **Configure:** The host selects character packs/generation and game mode (*Classic* or *Rule Master*).
4. **Play!** A shared grid of 24 characters is dealt. Click to eliminate characters from your local view. Ask your opponent questions over voice chat until only the winner remains.

## 💻 Local Development

Want to run this locally or contribute? Here is how to set it up:

### Prerequisites
* Node.js (v18+ recommended)
* A free [Pusher](https://pusher.com/) account for WebSocket keys.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/brayannhc07/anime-guess-web.git
   cd anime-guess-web
   ```

2. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```

3. Create `frontend/.env.local` with your Pusher credentials:
   ```env
   PUSHER_APP_ID=your_app_id
   PUSHER_KEY=your_key
   PUSHER_SECRET=your_secret
   PUSHER_CLUSTER=your_cluster
   NEXT_PUBLIC_PUSHER_KEY=your_key
   NEXT_PUBLIC_PUSHER_CLUSTER=your_cluster
   ```

4. Start the dev server:
   ```bash
   npm run dev
   ```

### Commands

```bash
cd frontend
npm run dev      # Start dev server (Next.js)
npm run build    # Production build
npm run lint     # ESLint
```
