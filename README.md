# 🕵️‍♂️ Adivinamon (Guess Who: Pokémon Edition)

A real-time multiplayer web game that brings the classic tabletop experience of "Guess Who?" (Adivina Quién) to the Pokémon universe. Connect with a friend via a room code, hop on Discord, and test your Pokémon knowledge to deduce their secret pick!

## ✨ Features

* **Real-Time Multiplayer:** Create or join a room using a unique 5-character code. State is synced instantly.
* **Two Game Modes:**
  * **Classic Guess Who:** Pick a secret Pokémon and ask yes/no questions to narrow down the opponent's choice.
  * **Rule Master:** Create a secret rule (e.g., "Only Pokémon with wings") and figure out the opponent's rule through strategic guessing.
* **Custom Generations:** Play with Gen 1 only, Gen 1-3, Gen 1-6, or pure chaos (all Pokémon).
* **Smart Caching:** Powered by TanStack Query and PokeAPI to load sprites instantly without hitting API rate limits.
* **Honor System UI:** Built to be played while chatting over Discord. You handle the talking, the app handles the board state.

## 🛠️ Tech Stack

* **Frontend:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/)
* **Local State:** [Zustand](https://github.com/pmndrs/zustand)
* **Data Fetching:** [TanStack Query](https://tanstack.com/query/latest) & [PokeAPI](https://pokeapi.co/)
* **WebSockets / Multiplayer:** [Pusher](https://pusher.com/) (or PartyKit)

## 🎮 How to Play

1. **Host a Game:** Player 1 clicks "Create Room" and selects the Pokémon Generation preset.
2. **Join a Game:** Player 2 enters the 5-character Room Code generated by the Host.
3. **Select Mode:** Agree on playing either *Classic* or *Rule Master*.
4. **Play!** Use your mouse to click and eliminate Pokémon from your local grid. Ask your opponent questions over voice chat until only the winner remains.
